Rowjam Example in Browser

All the fun stuff is happening in the JavaScript console. Open it up to see.

To run the example the first time

  1. Open a terminal, change to the example_browser directory.
  2. run ./build.sh. This should install the needed npm modules, run browserify to combine all the js files into "bundle.js", and open the HTML page in the browser.
  3. Open the javascript window in the browser.
